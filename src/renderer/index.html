Welcome to Cognition WP

The open-source, extensible word processor. Start typing to begin your document.

Cognition WP is an open-source, extensible word processor built on Electron and TypeScript. It features a full extension system for adding new capabilities.

Key Features

  • Rich text editing with full formatting toolbar
  • Extension system for plugins and add-ons
  • Multiple themes (Light, Dark, Sepia, High Contrast)
  • Markdown import/export support
  • Command palette (Ctrl+Shift+P)
  • Full keyboard navigation
Tip: Press Ctrl+Shift+P to open the command palette and discover everything Cognition WP can do.

Press Ctrl+N for a new document, or just start editing this one.
